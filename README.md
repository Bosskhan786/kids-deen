# 🌙 Kids Deen

A simple and engaging Islamic learning platform for kids — designed to teach basic Deen through interactive content like Hadith, duas, and stories.

## 🚀 Live Website

👉 https://kids-deen.vercel.app

## 📂 GitHub Repository

👉 https://github.com/Bosskhan786/kids-deen

---

## 📌 Features

* 📖 Daily Hadith for kids
* 🤲 Simple Duas with meanings
* 🧠 Easy-to-understand Islamic teachings
* 🎨 Clean and kid-friendly UI
* ⚡ Fast and responsive design

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Deployment: Vercel
* Version Control: Git & GitHub

## 💡 Future Improvements
